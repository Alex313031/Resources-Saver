Instructions

  • This extension works in the devtools environment, so it can read the resources information. You will need to restart your browser or re-open the devtools panel, and switch to the Resources Saver tab.
  • All resources will be saved in the "Downloads" folder as (<DOMAIN>).
  • Tick "Download from all domains ..." to download all resources from all domains.
  • If "Compress all assets ... (Recommended)" is ticked, all resources will be added into one compressed file as (<DOMAIN>.zip).
  • If you want to manually save the file(s), enable the option in Main Menu > Settings > Downloads > "Ask where to save each file before downloading".
